# Chorify 商业计划书

> 文档性质：**软件开发导向的商业计划书**。既说明"为什么做、做给谁、靠什么活下去"，也为后续产品与技术文档（02 / 03 / 04）提供商业前提。
> 阶段定位：**天使 / 种子轮**。先以一个"真骨架 + AI 能力可插拔"的 MVP 验证 PMF。

---

## 1. 项目概述

**Chorify** 是一个一站式 **AI 视频营销平台**，把"看灵感 → 出素材 → 批量成片 → 多号分发 → 复盘运营"这条原本需要一个内容团队数天完成的链路，压缩成一个人、一个下午、在一个网页里完成。

- **一句话**：让每一个商家都能像流水线一样，工业化、规模化地生产并分发爆款营销视频。
- **核心链路**：对标筷子科技的"**编拍剪投管**"，在 Chorify 中落为
  **编导灵感 → 素材生产 → 合成量产 → 投放分发**（工作流）+ **云盘 / 手机协同 / 直播切片 / 视频翻译 / 矩阵宝 / BGM市场 / 视频拆分**（服务）+ **组织 / 团队 / 计费**（管理底座）。
- **形态**：Web SaaS（多租户），AI 能力以独立服务可插拔接入，先 mock 后接真实供应商。
- **阶段目标**：用最小成本跑通"商家从 0 到批量出片再到分发"的闭环，拿到首批付费验证。

---

## 2. 行业背景与趋势

1. **视频已成商业流量的默认语言。** 短视频 + 直播是电商、本地生活、品牌种草的主战场，平台（抖音 / 快手 / 视频号 / TikTok / YouTube Shorts）的分发逻辑高度偏好"量大、更新快、多账号矩阵"的内容供给。
2. **内容供给侧严重产能不足。** 一条人工短视频从脚本、拍摄、剪辑到适配多平台，成本高、周期长、难规模化；而投放算法需要的是"千条起步、持续迭代"的素材量。**供需缺口正是 AIGC 的机会。**
3. **AIGC 让"工业化产能"第一次成为可能。** 文本大模型写脚本、文生图 / 图生视频生成素材、TTS 与数字人替代真人出镜、智能混剪把少量优质素材裂变成上千条组合——内容生产从"手工作坊"走向"流水线工厂"。
4. **赛道已被验证。** 头部玩家筷子科技（见第 4 节）已服务全球 8500+ 企业、累计生成分发 8000 万+ 商业视频，并在 2025 年完成近亿元 B 轮、启动数亿元 C 轮——证明"AI 视频营销工业化"是真实且可持续付费的市场。
5. **空白仍在。** 头部主要服务大中型品牌与集团客户；**海量中小商家、本地生活门店、跨境出海个体卖家**仍被"贵、重、难上手"挡在门外。这是 Chorify 的切入口。

> 注：本文涉及的行业规模为公开报道与行业研究的方向性引用，用于判断赛道量级与趋势，非精确财测；正式融资材料需以最新第三方数据复核。

## 3. 市场与目标客户

### 3.1 目标客户分层

| 优先级 | 客群 | 画像与痛点 | Chorify 价值 |
|---|---|---|---|
| **P0（冷启动）** | 中小电商 / 白牌商家 | 缺内容团队、要海量带货短视频喂投放 | 一键批量混剪，几十块成本出上百条 |
| **P0** | 本地生活门店 / 连锁 | 门店多、要同城短视频矩阵铺量 | 模板化批量出片 + 矩阵宝多号分发 |
| **P1** | 跨境出海卖家 | 一套素材要多语言适配多平台 | 视频翻译 + 多语言配音 + 多平台分发 |
| **P1** | MCN / 代运营机构 | 同时服务几十个客户，人效是生死线 | 组织协同 + 模板复用 + 批量产能 |
| **P2** | 品牌市场部 | 要素材合规、品牌一致、可复盘 | 云盘资产沉淀 + 团队权限 + 数据看板 |

### 3.2 市场规模（量级判断，自上而下 + 自下而上）

- **TAM**：中国数字营销 / 内容营销整体市场（万亿级人民币量级）中，与"视频内容生产 + 分发投放"强相关的部分。
- **SAM**：愿意为"AI 视频内容工业化生产工具"付费的商家与机构（数百万级商家基数 × 年均工具/服务付费）。
- **SOM（3 年可触达）**：以中小商家、本地生活、出海卖家为核心客群，通过 PLG（免费试用转化）+ 渠道/代运营分销获取的可服务份额。

> 量级用于判断"赛道足够大、值得投入"，具体数字在融资 PR 阶段以第三方报告锚定。Chorify 早期不赌总量，赌的是**头部尚未吃透的中小长尾**。

## 4. 对标分析：筷子科技（Kuaizi.ai）

> Chorify 在产品结构上**显式对标筷子科技**，因此先把对标对象讲透，再讲我们与它的异同。以下为公开资料整理。

### 4.1 对标对象画像

- **主体**：广州筷子信息科技有限公司；平台 **2015 年上线**。
- **创始人**：陈万锋，曾任职宝洁（P&G）；天使投资人包括曾李青（腾讯联合创始人）、刘成敏（腾讯前高级执行副总裁）。
- **定位**：**AI 内容商业应用智能体平台**——用生成式 AI 为全球品牌与商家提供内容商业解决方案，围绕"**编拍剪投管**"核心链路构建一体化 AIGC 平台。
- **规模（公开口径）**：服务全球 **8500+ 企业**、覆盖 **800+ 城市**、累计生成分发 **8000 万+ 商业视频**。
- **典型案例**：八合里牛肉火锅用"短视频批量混剪 + 矩阵宝"，覆盖 200+ 城市门店、月分发 800+ 视频、累计生产 13000+ 视频。

### 4.2 融资轨迹（公开报道）

| 轮次 | 金额 | 投资方（部分） |
|---|---|---|
| 天使 | — | 曾李青、刘成敏 |
| Pre-A | 1080 万 | 启赋资本领投，蔡晓东跟投 |
| A+ | 数千万 | 赛意产业基金 |
| B1 | 近 5000 万 | 金沙江创投、海珠数字、明辉懿德、海南比芯（非凡资本任 FA）|
| **B** | **近亿元（2025.6）** | 用于内容商业 AI 智能体与数据基础设施、全球化布局（Kuaizi 5.0）|
| C | 数亿元（已启动） | — |

### 4.3 筷子的"编拍剪投管"五段链路

| 段 | 模块 | 能力关键词 |
|---|---|---|
| **编** | 编导灵感 | 全网灵感搜索/洞察、爆款视频解构、AI 创作、智能 Agent 助理「小K」|
| **拍** | 素材生产 | 多模态生成：文本/图片/视频/音频/数字人，AI 影棚、AI 复刻、元素替换、多语言配音 |
| **剪** | 合成量产 | 短视频智能混剪、一键批量生成上千条、直播流理解自动出片、超级混剪 Pro |
| **投** | 投放分发 | 矩阵宝多平台多号分发、公域分发、二维码/KOC 视频卡片、智能评论与私信 |
| **管** | 管理运营 | 角色资源分配、团队效能、全局数据统筹 |

### 4.4 Chorify 与筷子的异同

- **学什么（结构对齐）**：完整复用"工作流 + 服务 + 管理"的产品骨架与"编拍剪投管"链路心智——这是被市场验证过的正确结构。
- **不同点（差异化定位）**：
  1. **客群下沉**——筷子重心在品牌/集团大客户；Chorify 先吃中小商家与本地生活、出海个体卖家。
  2. **更轻更快上手**——弱化复杂配置，主打"模板 + 一键"，价格亲民、按量付费。
  3. **Agent 原生**——把"小助手"做成贯穿全流程的入口（从灵感到分发都能对话驱动），而非附加功能。
  4. **能力可插拔 + 数据自主**——AI 供应商可替换、支持私有化/数据导出，降低客户锁定顾虑。
  5. **出海友好**——多语言配音/翻译 + 海外平台（TikTok/YouTube）一等公民。

## 5. 痛点与解决方案

| 商家痛点 | 传统做法 | Chorify 方案 | 对应模块 |
|---|---|---|---|
| 不知道拍什么 | 凭感觉/抄同行 | AI 全网灵感洞察 + 爆款解构 | 编导灵感 |
| 没素材、请不起团队 | 外包/自拍，贵且慢 | AI 文/图/视频/配音/数字人生成 | 素材生产 |
| 投放要海量素材 | 人工剪辑，产能跟不上 | 智能混剪一键出上百上千条 | 合成量产 |

| 出海要本地化 | 重新做一遍 | 视频翻译 + 多语言配音 | 视频翻译 |
| 不知道哪条有效 | 各平台后台割裂 | 分发数据统一看板 | 投放分发 / 管理 |

## 7. 商业模式与定价

### 7.1 收入来源

1. **订阅（主）**：按席位 + 功能档（Free / Pro / Enterprise，已在数据模型 `Plan` 中预留）。
2. **用量计费（叠加）**：AI 重操作（视频生成、数字人、配音分钟数、批量混剪条数）按 **积分/Credits** 扣减——直接对应可变的供应商成本，保证毛利。
3. **增值服务**：私有化部署、定制模型/数字人、代运营与陪跑（面向 MCN/品牌）。

### 7.2 定价档位（草案）

| 档位 | 目标客群 | 形态 | 包含 |
|---|---|---|---|
| Free | 个人/试用 | 免费 | 限量 Credits、水印、基础模板 |
| Pro | 中小商家 | 月/年订阅 + Credits 包 | 全功能、去水印、矩阵宝有限账号 |
| Enterprise | 机构/品牌 | 年签 + 用量 | 组织协同、SSO、私有化、专属客服 |

### 7.3 成本结构（与定价直接挂钩）

- **可变成本**：AI 供应商调用（LLM / 文生视频 / TTS / 数字人）、渲染算力、OSS 存储与流量、短信。→ 通过 Credits 透传，保住毛利。
- **固定成本**：研发团队、基础设施、获客。
- **关键经营指标**：单条视频生产成本、付费转化率、Credits 复购率、客户留存（月活/续费）。

## 8. 竞争格局与差异化

- **直接对标**：筷子科技（结构最接近、最值得学习）。
- **生态对手**：剪映 / 即梦（字节系）、可灵（快手系）等——强在单点生成能力与平台流量，但偏"创作工具"，弱在"商家工业化 + 多号分发 + 组织协同"的闭环。
- **替代方案**：人工剪辑团队、外包、各平台自带工具——慢、贵、难规模化。

**Chorify 的护城河（逐步构建）**：
1. **链路闭环**——别人给工具，我们给"从灵感到分发"的完整生产线。
2. **场景模板沉淀**——按行业（餐饮/美妆/3C/本地生活/跨境）积累可复用模板与最佳实践。
3. **数据飞轮**——分发回流的效果数据反哺选题与混剪策略。
4. **Agent 体验**——对话式驱动降低使用门槛，提升留存。
5. **成本工程**——多供应商可插拔 + 缓存/批处理，做到比对手更低的单条成本。

## 9. 融资与预算（天使 / 种子轮）

> 本阶段融资的本质是"**买时间把 MVP 跑通并拿到首批付费验证**"，预算口径以研发与算力为主。

### 9.1 资金用途

| 方向 | 占比（建议） | 说明 |
|---|---|---|
| 研发团队 | ~55% | 全栈 + AI 工程 + 产品/设计（见第 10 节） |
| AI 算力 / 供应商 | ~20% | LLM、文生视频、TTS、数字人调用与渲染 |
| 基础设施 | ~10% | 服务器、PostgreSQL/Redis、阿里云 OSS、短信、监控 |
| 获客与运营 | ~10% | 种子客户获取、内容/案例、渠道试点 |
| 机动 | ~5% | 合规、法务、备用 |

### 9.2 里程碑式资金消耗（与路线图对齐）

- **M0–M3**：MVP 真骨架（已部分完成）+ 接通首个真实 AI 供应商，拿到 10–20 个种子商家试用。
- **M3–M6**：合成量产 + 投放分发跑通闭环，首批付费转化，验证单条成本与毛利。
- **M6–M12**：服务模块补齐（翻译/直播切片/矩阵宝）+ 出海场景，冲击下一轮所需的留存与收入数据。

### 9.3 关键验证指标（投资人关心、也是产品北极星）

- 周活商家数、人均出片量、付费转化率、Credits 复购、单条视频生产成本、30 日留存。

## 11. 风险与对策

| 风险 | 说明 | 对策 |
|---|---|---|
| 巨头碾压 | 字节/快手系工具流量与模型强 | 聚焦"商家工业化闭环 + 组织协同"差异化，避开纯创作工具正面战 |
| AI 成本不可控 | 供应商涨价/调用贵 | Credits 透传成本、多供应商可插拔、缓存与批处理 |
| 内容合规 | 生成内容 / 分发违规风险 | 审核流水线、平台规则适配、水印与溯源 |
| 平台política变动 | 分发平台 API/政策收紧 | 抽象分发适配层，单平台变动不影响主链路 |
| 同质化 | 结构易被模仿 | 用模板沉淀 + 数据飞轮 + 服务体验筑墙 |
